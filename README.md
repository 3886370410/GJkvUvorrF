# 项目简介

## 核心功能
- 集成百度千帆AI大模型
- 可视化大屏功能
- 首页活动信息推荐采用协同过滤算法
- 学生功能实现
  - 活动信息推荐
  - 活动信息
  - 活动论坛
  - 留言板
  - 活动报名
- 教师功能实现
  - 活动信息
  - 活动报名
  - 活动评价
- 管理员功能实现
  - 教师管理
  - 用户管理
  - 活动信息
  - 活动报名
  - 活动评价
  - 公告信息
  - 活动论坛
  - 留言板

## 适用场景
- 学生课外活动管理
- 教师活动发布与评价
- 管理员活动管理与维护

## 技术特点
- B/S架构，支持Web访问
- MVC模式，便于开发与维护
- Java后端，SpringBoot框架
- Vue前端，响应式设计
- 使用Maven进行项目构建
- 数据库支持MySQL
- 支持IDEA开发环境
